## Mnemonic Memory System

No system prompt required. Mnemonic's tools are self-describing — each tool includes
"use when" / "do not use when" guidance, behavioral annotations, and typed schemas.

For on-demand workflow guidance, use the `mnemonic-workflow-hint` MCP prompt. It covers
the discover → inspect → modify → organize pattern, the storage-label model, and the
`recall` → `get` → `update` preference.
