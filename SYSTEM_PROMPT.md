## Mnemonic Memory System

No system prompt required. Mnemonic's tools are self-describing — each tool includes
"use when" / "do not use when" guidance, behavioral annotations, and typed schemas.

If your model isn't picking up the tools proactively, add this single line to your
AGENT.md or system prompt:

> You have access to a long-term memory system via the `mnemonic` MCP server. Use it proactively — don't wait to be asked.
